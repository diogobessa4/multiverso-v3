# Pasta de fila de cadastro

Esta pasta é usada como fila local pelo script
`scripts/cadastrar-funkos.js`. Coloque aqui as imagens dos Funkos
que quer cadastrar e rode o script. Após cadastro bem-sucedido,
as imagens são movidas para `imagens/cadastrados/`.

**Nada nesta pasta é commitado** — está no `.gitignore`.
`.gitkeep` e `README.md` são as únicas exceções.
